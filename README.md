# MUTEACH
Manage your students, sesions, attendance and evaluations.
https://www.muteach.com

## Setup
1. Install Ruby
```bash
rvm install (whatever version within Gemfile)
gem install bundler
bundle
```

2. Install Yarn
```bash
brew install yarn
# Install node packages
yarn
```

3. Create DB
```bash
rails db:create db:migrate
```

4. Run local server
```bash
rails s
```

5. Run Webpack
```bash
yarn start
```
